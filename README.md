# OpenClaw Voice

Local TTS with custom voice cloning for [OpenClaw](https://github.com/openclaw/openclaw) agents.

## Setup

Tell your agent:

> "Set up voice capabilities"

or

> "I want to add a custom voice"

The agent will handle installation, configuration, and guide you through adding custom voices.

## Responsible Use

This tool enables voice cloning. Use responsibly — get consent before cloning someone's voice, don't use for impersonation or fraud.
